The CPU is very important for a computer because it works as the computer's brain. It is a key factor in terms of the speed performance of the computer. The CPU is basically a very small chip responsible for calculations as well as the responsibility of making all the logical decisions, which in fact is every single task on the computer. Tasks can be anything from typing a letter on your keyboard, opening and playing music on Spotify or doing a Google search.

The CPU has three main functionalities. First it receives some input data based on an action, this could be for example opening a text editor. The CPU then calculates and understands the data and processes it. Finally it returns and provides the output data.

The socket in a CPU works as the binding part of the motherboard and the CPU. The socket is providing the CPU with electricity.

The cores are responsible for the speed performance of the CPU. This means that the more cores the CPU has - the quickly it can perform tasks. The amount of cores also decides how many tasks can be processed simultaneously.

Cache works as the memory of the CPU. Data that is accessed often will be saved in this part of the CPU to improve the speed performance for when accessing this data.

The part of the computer where all the data is stored. This part of the computer is essential for the entire computer to save anything, but this is also where the CPU accesses the data.

Keyboard, mouse, monitor ect. Items that can do actions that require the CPU to do tasks.

The clock of the CPU is a key factor to the speed performance of the CPU. The CPU clock is measured in GHz. For each GHz the CPU can perform 1.000.000.000 clock cycles per second.

So if the CPU has 2.6 GHz it will be able to do 2.600.000.000 clock cycles per second!
